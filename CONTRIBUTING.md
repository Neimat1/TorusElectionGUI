# Contributing

Thank you for considering a contribution to Torus Election GUI.

## Development Setup

1. Install JDK 17 or later.
2. Install Maven 3.8 or later.
3. Run the test suite:

```bash
mvn test
```

## Contribution Guidelines

- Keep changes focused and consistent with the existing package structure.
- Add or update tests when behavior changes.
- Run `mvn test` before opening a pull request.
- Update documentation when user-facing behavior, build steps, or packaging changes.
